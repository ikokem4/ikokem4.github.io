AI 검색이 기존 검색을 대체할 수 있을까 — Google SGE, Perplexity 이야기

AI 검색 서비스인 Google SGE와 Perplexity를 비교하며, 기존 검색 방식이 어떻게 바뀌고 있는지 정리했습니다.

검색하는 방식이 바뀌고 있다#

예전에는 뭔가 궁금하면 구글에 키워드를 넣고, 나오는 링크 10개 중에서 괜찮은 걸 골라서 클릭하는 게 당연했잖아요. 근데 요즘은 AI한테 바로 물어보는 게 더 빠를 때가 많습니다.

“RTX 4070 Super vs RTX 5070 차이가 뭐야?“라고 치면 블로그 5개를 돌아다닐 필요 없이 AI가 바로 정리해서 알려주거든요.

이런 흐름에서 나온 게 Google SGE(AI 개요)와 Perplexity입니다. 둘 다 써봤는데, 생각보다 느낌이 다르더라고요.

Google SGE (AI 개요) — 검색 결과 위에 AI 답변이 뜬다#

구글에서 뭔가 검색하면 요즘 검색 결과 맨 위에 AI가 정리한 답변이 뜨는 경우가 있습니다. 이게 SGE(Search Generative Experience), 지금은 “AI 개요"라고 부르는 기능이에요.

기존 검색 방식을 완전히 바꾸는 건 아니고, 기존 검색 결과 위에 AI 요약이 추가된 형태입니다.

장점:

  • 기존 구글 검색 흐름 그대로 유지
  • 따로 앱을 깔거나 사이트에 갈 필요 없음
  • 참고 출처 링크가 같이 나옴

아쉬운 점:

  • 모든 검색어에 뜨는 건 아님
  • 답변 품질이 들쭉날쭉할 때가 있음
  • 광고와 AI 답변이 섞여서 나올 때 좀 산만함

솔직히 말하면, 간단한 사실 확인 정도는 괜찮은데 좀 복잡한 비교나 심층 분석은 아직 부족한 느낌입니다.

Perplexity — 검색 전용 AI#

Perplexity는 아예 처음부터 “AI 검색 엔진"을 표방하고 나온 서비스입니다. 질문을 하면 웹을 검색하고, 결과를 종합해서 답변을 만들어주는 방식이에요.

써보면 느끼는 게, 구글보다 답변이 좀 더 깔끔하게 정리되어 있습니다. 출처도 번호로 달려 있어서 어디서 가져온 정보인지 바로 확인할 수 있고요.

장점:

  • 답변 품질이 일관적으로 높음
  • 출처 표시가 깔끔함
  • 후속 질문으로 대화를 이어갈 수 있음
  • Pro 모드에서는 GPT-4, Claude 등 여러 모델 선택 가능

아쉬운 점:

  • 무료 버전은 하루 질문 수 제한 있음
  • Pro 버전은 월 20달러
  • 최신 정보가 반영 안 되는 경우도 있음
  • 한국어 검색은 영어보다 품질이 떨어짐

개인적으로 느끼는 건, Perplexity가 “검색을 위한 AI"로서는 지금까지 나온 것 중에 제일 잘 만들어진 것 같습니다.

기존 구글 검색 vs AI 검색 — 뭐가 다른 건가#

근본적인 차이는 정보를 찾아주느냐, 답을 만들어주느냐입니다.

구글 검색은 관련 페이지를 찾아서 보여주는 거고, AI 검색은 여러 소스를 읽고 종합해서 하나의 답변을 만들어주는 거예요.

이게 편하긴 한데, 리스크도 있습니다:

  • AI가 정보를 잘못 종합할 수 있음 (할루시네이션)
  • 출처를 확인 안 하면 틀린 정보를 맞다고 믿게 됨
  • 원본 콘텐츠 제작자에게 트래픽이 안 감

특히 마지막 문제가 꽤 큰데, 블로그나 뉴스 사이트 입장에서는 자기가 쓴 글을 AI가 요약해서 보여주니까 방문자가 줄어드는 거잖아요. 이 부분은 앞으로 계속 논란이 될 것 같습니다.

어떤 상황에서 뭘 쓰면 좋을까#

용도에 따라 다르게 쓰는 게 맞는 것 같습니다.

기존 구글 검색이 나은 경우:

  • 특정 사이트나 페이지를 찾고 싶을 때
  • 쇼핑, 지도, 이미지 검색
  • 최신 뉴스 확인

AI 검색(Perplexity 등)이 나은 경우:

  • 개념 비교 (“A vs B 차이”)
  • 복잡한 질문의 종합 답변
  • 여러 소스를 한번에 정리하고 싶을 때
  • 리서치 초기 단계에서 방향 잡을 때

저는 요즘 리서치할 때 Perplexity로 먼저 큰 그림을 잡고, 세부 내용은 구글에서 원본 자료를 찾아보는 식으로 쓰고 있습니다. 이렇게 섞어 쓰는 게 지금으로서는 제일 효율적인 것 같아요.

AI 검색이 구글을 대체할 수 있을까#

솔직히, 완전히 대체하기는 어려울 것 같습니다. 적어도 당분간은요.

구글 검색은 단순히 텍스트 답변만 주는 게 아니라 지도, 쇼핑, 이미지, 뉴스, 유튜브까지 통합된 플랫폼이거든요. AI 검색이 아무리 답변을 잘 해줘도 이 생태계를 대체하기는 쉽지 않습니다.

하지만 “정보를 검색하는 방식” 자체는 확실히 바뀌고 있습니다. 링크 10개 보고 골라 클릭하는 시대에서, AI한테 물어보고 바로 답 받는 시대로 넘어가고 있는 건 맞아요.

앞으로 1~2년 안에 AI 검색이 지금보다 훨씬 자연스러워질 거고, 구글도 SGE를 계속 강화할 겁니다. 검색의 형태가 바뀌는 과도기에 있다는 건 확실합니다.
