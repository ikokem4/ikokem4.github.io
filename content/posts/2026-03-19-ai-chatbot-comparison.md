## 다 비슷해 보이는데 뭐가 다른 건지

AI 챗봇이 하나둘 나올 때는 ChatGPT만 쓰면 됐습니다. 근데 이제 Claude도 있고, Gemini도 있고, 선택지가 많아지니까 오히려 뭘 써야 할지 모르겠다는 분들이 많더라고요.

세 개 다 써본 입장에서 말씀드리면, **용도에 따라 확실히 차이가 있습니다.** 전부 잘하는 만능 AI는 아직 없고, 각자 잘하는 영역이 다릅니다.

## 각각 뭘 잘하나

### ChatGPT (OpenAI)

가장 먼저 대중화된 AI라서 사용자가 압도적으로 많습니다. 생태계가 넓은 게 가장 큰 강점이에요.

**잘하는 것:**
- 플러그인/GPTs가 엄청 많음 — PDF 분석, 데이터 시각화, 이미지 생성까지 한 곳에서 가능
- DALL-E 이미지 생성이 내장되어 있음
- 대화가 자연스럽고 범용적
- 웹 브라우징 기능으로 최신 정보 검색 가능

**아쉬운 것:**
- 긴 문서를 넣으면 중간 내용을 빼먹는 경우가 있음
- 무료 버전은 기능 제한이 꽤 있음
- 가끔 자신있는 어조로 틀린 말을 함

일반적인 질문, 이미지 생성, 다양한 플러그인 활용이 필요하면 ChatGPT가 편합니다.

### Claude (Anthropic)

제가 이 블로그를 운영하는 데 실제로 쓰고 있는 AI입니다. 긴 텍스트 처리와 코딩에서 강점이 뚜렷해요.

**잘하는 것:**
- 긴 문서 처리 능력이 뛰어남 — 책 한 권 분량도 한번에 읽고 분석
- 코딩 능력이 좋음 — 특히 코드를 직접 수정하고 테스트하는 에이전트 기능
- 지시사항을 정확하게 따르는 편
- 모르는 건 모른다고 말하는 경향 — [환각](/posts/2026-02-27-ai-hallucination/) 문제가 상대적으로 적음

**아쉬운 것:**
- 이미지 생성 기능이 없음
- 웹 검색 기능이 제한적
- ChatGPT에 비해 플러그인/확장 기능이 적음

보고서 분석, 긴 문서 요약, 코딩 작업에는 Claude가 체감상 가장 좋습니다.

### Gemini (Google)

구글이 만든 AI입니다. 구글 생태계와 연결되는 게 핵심 강점이에요.

**잘하는 것:**
- 구글 검색과 통합 — 최신 정보 접근이 가장 빠름
- Gmail, Google Docs, Drive 등 구글 서비스와 연동
- 이미지 이해 능력이 좋음
- 무료로도 꽤 많은 기능 사용 가능

**아쉬운 것:**
- 한국어 답변 품질이 ChatGPT나 Claude보다 떨어지는 경우가 있음
- 긴 대화에서 맥락을 놓치는 경우가 종종 있음
- 코딩 능력은 다른 둘에 비해 조금 아쉬움

구글 서비스를 많이 쓰는 분이라면 Gemini가 연동 측면에서 편리합니다.

## 용도별로 비교하면

| 용도 | 추천 | 이유 |
|------|------|------|
| 일반 질문/대화 | ChatGPT | 가장 범용적, 대화가 자연스러움 |
| 긴 문서 분석/요약 | Claude | 긴 컨텍스트 처리 능력 최고 |
| 코딩/개발 | Claude | 코드 수정·테스트까지 직접 수행 |
| 이미지 생성 | ChatGPT | DALL-E 내장 |
| 최신 정보 검색 | Gemini | 구글 검색 통합 |
| 구글 서비스 연동 | Gemini | Gmail, Docs, Drive 직접 연동 |
| 영어 학습 | ChatGPT/Claude | 둘 다 좋음, [활용법은 여기](/posts/2026-03-18-ai-english-study/) |
| 프라이버시 중시 | Claude | 대화 내용을 학습에 사용하지 않는 옵션 |

## 유료 결제할 가치가 있나

세 서비스 모두 무료로 쓸 수 있지만, 무료 버전은 기능이 제한됩니다.

**ChatGPT Plus (월 $20)** — GPT-4o 무제한, DALL-E, 플러그인 전체 사용. AI를 매일 다양하게 쓴다면 가성비가 좋음.

**Claude Pro (월 $20)** — 최신 모델 사용량 증가, 긴 문서 처리 한도 확대. 업무용으로 문서나 코드를 많이 다루면 값어치를 함.

**Gemini Advanced (월 $20)** — 최신 모델 + 구글 서비스 심화 연동. 구글 워크스페이스를 업무에 쓰는 분에게 유리.

셋 다 가격이 같으니까, **본인이 가장 많이 쓰는 용도**에 맞는 걸 하나만 결제하는 게 효율적입니다. 셋 다 결제할 필요는 전혀 없어요.

## 하나만 골라야 한다면

딱 하나만 고르라고 하면 저는 이렇게 추천합니다.

- **뭘 골라야 할지 모르겠다** → ChatGPT. 범용적이고 할 수 있는 게 가장 많음
- **업무/생산성 위주** → Claude. 문서 처리와 정확도가 좋음
- **구글 서비스 헤비 유저** → Gemini. 연동의 편리함이 큼

근데 솔직히, 무료 버전으로 세 개 다 써보고 본인한테 맞는 걸 고르는 게 가장 확실합니다. 글로 아무리 비교해봐야 직접 써보는 것만 못하거든요. 세 서비스 다 가입은 무료니까, 같은 질문을 셋한테 던져보면 차이가 바로 느껴질 겁니다.
