## USB가 원래 이렇게 복잡했나

USB는 **Universal Serial Bus**의 약자로, 원래 "하나로 통일하자"는 취지에서 만들어졌습니다. 근데 지금은 USB-A, USB-C, USB 2.0, 3.0, 3.1, 3.2, 4.0, 썬더볼트... 오히려 더 복잡해졌습니다.

저도 케이블 살 때마다 헷갈려서, 한 번 정리해봤습니다.

## 먼저 모양(커넥터)과 속도(규격)를 구분하자

헷갈리는 이유는 **모양**과 **속도**가 별개이기 때문입니다.

### 모양 (커넥터 타입)

- **USB-A** — 우리가 아는 직사각형 USB. 방향 맞춰서 꽂아야 함
- **USB-C** — 양면 다 되는 타원형. 요즘 표준
- **Micro USB** — 예전 안드로이드 폰 충전 단자. 이제 거의 안 씀

### 속도 (규격 버전)

| 규격 | 최대 속도 | 쉬운 이름 |
|------|-----------|-----------|
| USB 2.0 | 480Mbps | 느림 |
| USB 3.0 (3.2 Gen1) | 5Gbps | 보통 |
| USB 3.1 (3.2 Gen2) | 10Gbps | 빠름 |
| USB 3.2 Gen2x2 | 20Gbps | 매우 빠름 |
| USB4 | 40Gbps | 최고 |

이름이 계속 바뀌어서 짜증나는데, USB-IF(규격 관리 단체)가 이름을 여러 번 변경한 탓입니다. USB 3.0이 갑자기 USB 3.2 Gen1로 이름이 바뀌는 식이라 혼란스럽습니다.

## USB-C라고 다 같은 게 아니다

여기가 가장 많이 속는 부분입니다. **USB-C는 모양일 뿐**이고, 안에 들어있는 규격이 다를 수 있습니다.

같은 USB-C 포트인데:
- 하나는 **USB 2.0** 속도 (480Mbps)
- 하나는 **USB4** 속도 (40Gbps)

이런 경우가 진짜 있습니다. 특히 저가형 케이블이나 기기에서 USB-C인데 USB 2.0인 경우가 많습니다. 케이블을 살 때 모양만 보지 말고 규격을 확인해야 합니다.

## 썬더볼트는 뭐야

Intel과 Apple이 만든 고속 연결 규격입니다. USB-C 모양을 쓰지만 더 빠르고 기능이 많습니다.

- **Thunderbolt 3/4** — 40Gbps, USB-C 모양
- **Thunderbolt 5** — 80~120Gbps

모니터 연결, 외장 GPU 연결 같은 고급 기능은 썬더볼트가 필요한 경우가 있습니다. 포트 옆에 번개 마크(⚡)가 있으면 썬더볼트입니다. 썬더볼트에 대한 더 자세한 내용은 [썬더볼트 규격 완전 정리](/posts/2026-02-24-thunderbolt-explained/)를 참고해보세요.

## 충전도 USB마다 다르다

USB-C로 충전할 때도 규격에 따라 충전 속도가 다릅니다.

- **일반 USB** — 5W~15W (느린 충전)
- **USB PD (Power Delivery)** — 최대 240W (노트북 충전 가능)

노트북을 USB-C로 충전하려면 USB PD를 지원하는 충전기와 케이블이 필요합니다. 아무 USB-C 케이블이나 되는 건 아닙니다.

## 실생활에서 알면 좋은 것

1. **케이블이 중요** — 비싼 기기를 사도 케이블이 USB 2.0이면 느림
2. **파란색 포트 = USB 3.0 이상** — 보통 데스크탑에서 파란색 USB-A는 3.0
3. **외장 SSD 쓸 때** — USB 3.0 이상 포트에 꽂아야 제 속도가 나옴. [SSD 종류별 속도 차이](/posts/2026-03-08-ssd-types/)를 알면 더 이해가 빠릅니다
4. **데이터 전송용 vs 충전용 케이블** — 충전 전용 케이블은 데이터 전송이 안 됨

## 마치며

USB 규격이 복잡한 건 사실이지만, 핵심만 기억하면 됩니다:

- **모양(USB-C)과 속도(USB 3.x)는 별개**
- **USB-C라고 다 빠른 게 아님**
- **케이블도 규격 확인 필수**

이것만 알아도 케이블이나 기기 살 때 속는 일은 줄어들 겁니다.
