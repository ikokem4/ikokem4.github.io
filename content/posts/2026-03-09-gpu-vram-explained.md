## VRAM이 뭐야?

VRAM은 **Video RAM**의 줄임말로, GPU(그래픽카드)에 달린 전용 메모리입니다.

컴퓨터의 일반 RAM이 CPU가 쓰는 메모리라면, VRAM은 GPU가 쓰는 메모리입니다. GPU가 작업할 때 필요한 데이터를 여기에 올려놓고 씁니다.

## 쉽게 비유하면

- **GPU** = 요리사
- **VRAM** = 조리대
- **일반 RAM** = 냉장고

요리사(GPU)가 요리(연산)를 하려면 재료를 조리대(VRAM)에 올려놓아야 합니다. 조리대가 좁으면 재료를 다 올려놓지 못해서 냉장고(RAM)에서 왔다갔다 해야 하고, 그만큼 느려집니다.

조리대가 넓으면? 재료를 한 번에 펼쳐놓고 빠르게 요리할 수 있습니다.

## 게임할 때 VRAM

게임에서 VRAM은 주로 **텍스처(그래픽 데이터)**를 저장합니다.

- 4K 고해상도 텍스처 = VRAM 많이 필요
- 낮은 해상도/그래픽 설정 = VRAM 적게 필요

요즘 게임은 8GB 정도면 대부분 문제없고, 4K 울트라 설정이면 10~12GB가 편합니다.

## AI 작업할 때 VRAM

여기서 VRAM의 중요성이 확 올라갑니다. AI 모델은 통째로 VRAM에 올라가야 작동합니다.

### AI 모델 크기별 필요 VRAM

| 모델 크기 | 필요 VRAM | 예시 |
|-----------|-----------|------|
| 3~7B | 6~8GB | 가벼운 챗봇, 간단한 작업 |
| 8~14B | 10~16GB | 중급 작업, 코딩 보조 |
| 30~70B | 24~48GB | 고급 작업 |
| 70B 이상 | 48GB+ | 연구용 |

모델이 VRAM보다 크면 아예 실행이 안 되거나, 일반 RAM으로 넘겨서 처리하는데 이러면 엄청 느려집니다.

## 양자화하면 줄일 수 있다

"양자화(Quantization)"라는 기술을 쓰면 모델 크기를 줄여서 적은 VRAM으로도 돌릴 수 있습니다. 대신 성능이 약간 떨어집니다.

예를 들어 원래 16GB 필요한 모델을 양자화하면 8~10GB로 줄일 수 있습니다. 성능 손실은 있지만 체감상 크게 차이 안 나는 경우도 많습니다.

## VRAM은 나중에 추가할 수 있나?

**안 됩니다.** 이게 핵심입니다. 일반 RAM은 슬롯만 있으면 나중에 추가할 수 있지만, VRAM은 GPU에 붙어 있어서 바꿀 수 없습니다. GPU를 살 때 VRAM 용량을 잘 선택해야 하는 이유입니다.

## 현재 GPU별 VRAM 정리

| GPU | VRAM |
|-----|------|
| RTX 3060 | 12GB |
| RTX 4060 | 8GB |
| RTX 4070 | 12GB |
| RTX 4090 | 24GB |
| RTX 5070 | 12GB |
| RTX 5090 | 32GB |

재밌는 게, RTX 3060이 12GB인데 RTX 4060이 8GB입니다. 신형이 항상 VRAM이 많은 건 아니에요. AI 작업을 생각한다면 VRAM 용량을 꼭 확인해야 합니다.

## 마치며

정리하면:
- **게임** — VRAM 8GB면 대부분 OK
- **AI 작업** — VRAM이 많을수록 좋고, 최소 12GB 추천
- **VRAM은 추가 불가** — 처음에 잘 선택해야 함

GPU 스펙 볼 때 클럭 속도나 CUDA 코어 수보다 **VRAM 용량**을 먼저 확인하는 습관을 들이면 좋습니다.
